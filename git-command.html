Frequently Used Git Commands

[event]   [command]
delete a local branch    git branch -d my_local_branch_name
download a remote branch from different computers    git checkout -t origin/my-branch-name
clone a specific branch to local computer    git clone -b my-branch https://git@github.com/username/myproject.git
fetch a specific branch and merge it into current local branch    git pull [REMOTENAME] [BRANCHNAME]

add, commit, and push to remote github repo   
  • git add my-file-name
  • git commit -m "my comments here"
  • git remote add origin https://github.com/username/demo-repo.git
  • git push origin master

git basic 2   
    ------ commend only required once to define "upstream"
    ------ "upstream" is a local name for remote location
  • $ git remote add upstream https://github.com/malecki/edav.git
  • $ git remote add upstream https://github.com/tc2680/edav.git

    • ------ check current remote setting
  • $ git remote -v

    • ------ remove/then reset the remote location "upstream"
  • $ git remote rm upstream

    • ------ you can replace "rebase" with "merge"
  • $ git fetch upstream
  • $ git checkout gh-pages
  • $ git rebase upstream/gh-pages
